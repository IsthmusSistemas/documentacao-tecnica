# RelatorioExecucaoResponse
- **Namespace**: IsthmusWinthor.Dominio.Relatorios.Execucao
- **Nome do Arquivo**: RelatorioExecucaoResponse.cs

Este é um Data Transfer Object (DTO) utilizado para transportar informações sobre a execução de um relatório. Ele encapsula os dados que são retornados após a execução de um relatório, sem lógica de negócio complexa.
