# RelatorioListagem
- **Namespace**: IsthmusWinthor.Dominio.Relatorios.Execucao
- **Nome do Arquivo**: RelatorioListagem.cs

A classe **RelatorioListagem** é utilizada como um DTO (Data Transfer Object) para transporte de dados relacionados a relatórios de listagem no sistema. 

Essa classe possui propriedades anêmicas que são utilizadas para transmitir informações essenciais sobre um relatório específico, como seu identificador, nome, descrição e estado de atividade. Não possui lógica de negócios ou validações embutidas, sendo assim, sua principal função é facilitar a comunicação de dados entre as camadas da aplicação.
