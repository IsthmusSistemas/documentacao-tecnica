# AcoesIndex\<T\>

**Namespace**: IsthmusWinthor.Dominio.AzureSearchIndex  
**Nome do Arquivo**: AcoesIndex.cs  

> A `AcoesIndex<T>` é uma classe do tipo DTO/VIEWMODEL. Ela atua como um contêiner de transporte de dados, encapsulando uma lista de elementos tipados genéricos associados a um índice de pesquisa no Azure. Esta classe é utilizada para facilitar o transporte e serialização de dados que representam ações ou modificações em um índice de busca, tornando eficiente o trabalho com coleções de elementos indexáveis.
