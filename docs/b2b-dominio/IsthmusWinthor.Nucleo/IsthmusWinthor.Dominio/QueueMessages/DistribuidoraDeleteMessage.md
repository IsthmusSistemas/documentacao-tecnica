# DistribuidoraDeleteMessage
**Namespace**: IsthmusWinthor.Dominio.QueueMessages  
**Nome do Arquivo**: DistribuidoraDeleteMessage.cs  

DistribuidoraDeleteMessage é uma classe que serve como um objeto de transferência de dados (DTO) para encapsular a mensagem enviada ao sistema de filas, representando a solicitação de exclusão de uma distribuidora. Sua principal função é transportar a informação necessária para que o sistema reconheça qual distribuidora deve ser removida da base de dados.
