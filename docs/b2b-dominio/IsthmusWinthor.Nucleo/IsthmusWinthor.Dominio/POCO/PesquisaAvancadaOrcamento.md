# PesquisaAvancadaOrcamento
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaAvancadaOrcamento.cs  

A classe `PesquisaAvancadaOrcamento` é um Data Transfer Object (DTO) que serve como um modelo para transporte de dados em buscas relacionadas a orçamentos de clientes. Ela encapsula parâmetros que podem ser utilizados em consultas para filtrar informações específicas de orçamentos dentro de uma aplicação.
