# PesquisaClienteDesejo
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaClienteDesejo.cs  

Esta classe é um Data Transfer Object (DTO) utilizada para transporte de dados entre camadas do sistema, especificamente relacionada à pesquisa de desejos de clientes. Ela encapsula informações relevantes necessárias para executar consultas de produtos desejados por clientes, incluindo filtros como data, código do cliente, produto e opções de visualização. 

Observa-se que as propriedades são anêmicas, ou seja, não contêm lógica de negócio, não sendo necessário detalhar regras de negócio ou lógica adicional.
