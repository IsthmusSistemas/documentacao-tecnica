# ProdutoInformacoesIAMessage
**Namespace**: IsthmusWinthor.Dominio.POCO.PesquisaProdutos  
**Nome do Arquivo**: ProdutoInformacoesIAMessage.cs

O `ProdutoInformacoesIAMessage` é uma classe que serve como um modelo de transporte de dados (DTO) para transferir informações sobre produtos, especificamente em contextos relacionados a mensagens em fila. Ela é projetada para encapsular as informações que são relevantes para a pesquisa de produtos dentro de um domínio específico, permitindo a comunicação entre diferentes partes do sistema.

> Este é um DTO que transporta dados sobre informações de produtos, incluindo um identificador da distribuidora e detalhes do produto.
