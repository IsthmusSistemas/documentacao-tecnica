# BuscaProdutosOpenErpDocumento
**Namespace**: IsthmusWinthor.Dominio.POCO.PesquisaProdutos.OpenErp  
**Nome do Arquivo**: BuscaProdutosOpenErpDocumento.cs  

A classe `BuscaProdutosOpenErpDocumento` é um simples objeto de transporte de dados (DTO - Data Transfer Object) que encapsula a informação de um URL para um documento relacionado à busca de produtos no sistema Open ERP. Sua principal finalidade é facilitar a transferência dessa informação entre as camadas da aplicação, garantindo que a URL do documento seja manipulada de forma consistente.
