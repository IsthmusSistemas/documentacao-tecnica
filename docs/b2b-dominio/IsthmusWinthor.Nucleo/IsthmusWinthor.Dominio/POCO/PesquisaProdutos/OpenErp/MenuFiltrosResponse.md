# MenuFiltrosResponse
**Namespace**: IsthmusWinthor.Dominio.POCO.PesquisaProdutos.OpenErp  
**Nome do Arquivo**: MenuFiltrosResponse.cs  

O `MenuFiltrosResponse` é uma classe que serve como uma estrutura de transporte de dados (DTO) para encapsular informações relevantes sobre filtros de pesquisa de produtos, especificamente no contexto de um sistema ERP (Enterprise Resource Planning). Sua principal função é transportar dados relacionados a filtros de pesquisa, permitindo que diferentes camadas da aplicação se comuniquem de maneira eficaz.

### Citação
A classe `MenuFiltrosResponse` é utilizada para transportar dados relacionados aos filtros de um menu de pesquisa de produtos.
