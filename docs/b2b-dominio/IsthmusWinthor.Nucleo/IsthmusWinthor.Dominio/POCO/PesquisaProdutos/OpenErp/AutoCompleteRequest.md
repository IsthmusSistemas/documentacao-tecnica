# AutoCompleteRequest
**Namespace**: IsthmusWinthor.Dominio.POCO.PesquisaProdutos.OpenErp  
**Nome do Arquivo**: AutoCompleteRequest.cs  

Esta classe é utilizada como um objeto de transferência de dados (DTO) para facilitar a comunicação de informações entre diferentes camadas do sistema, especificamente em cenários de busca. Ela encapsula os parâmetros necessários para realizar uma requisição de autocompletar produtos com base em um termo de busca fornecido.
