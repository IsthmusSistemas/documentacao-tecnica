# FiltroConvenioPix
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: FiltroConvenioPix.cs  

A classe `FiltroConvenioPix` é utilizada como um objeto de transferência de dados (DTO) para carregar informações relacionadas ao filtro de convênios no sistema. Sua principal finalidade é transportar o código da filial que será utilizado em operações de filtro de convênios.

### Observação
Esta classe é um DTO/ViewModel, contendo apenas propriedades anêmicas e não incluindo regras de negócio ou lógica complexa.
