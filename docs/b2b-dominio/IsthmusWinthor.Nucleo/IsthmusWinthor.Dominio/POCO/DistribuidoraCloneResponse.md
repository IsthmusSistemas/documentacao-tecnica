# DistribuidoraCloneResponse
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: DistribuidoraCloneResponse.cs

Esta classe atua como um Data Transfer Object (DTO) que encapsula dados relacionados à resposta de uma operação de clonagem de uma distribuidora. Sua finalidade é transportar informações relevantes sobre a operação realizada, facilitando a comunicação entre diferentes partes do sistema.
