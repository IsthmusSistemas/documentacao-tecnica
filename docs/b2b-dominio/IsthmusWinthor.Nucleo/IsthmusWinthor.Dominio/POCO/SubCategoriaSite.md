# SubCategoriaSite
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: SubCategoriaSite.cs  

A classe `SubCategoriaSite` é um DTO (Data Transfer Object) cuja finalidade é transportar dados relacionados a subcategorias de um site. Ela contém propriedades que definem os atributos e características de uma subcategoria, permitindo a sua representação em interações entre camadas do sistema.
