# PesquisaAvancadaLoja
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaAvancadaLoja.cs  

A classe `PesquisaAvancadaLoja` é um DTO (Data Transfer Object) utilizado para o transporte de dados relacionados a pesquisas de lojas em um sistema corporativo. Sua finalidade é encapsular as informações necessárias para realizar uma pesquisa avançada, incluindo o nome da loja, a categoria e a opção de visualização desejada.

---

**Observação**: Não há regras de negócio, lógica de validação, nem propriedades calculadas ou navigation properties a serem documentadas neste caso, uma vez que se trata de um DTO.
