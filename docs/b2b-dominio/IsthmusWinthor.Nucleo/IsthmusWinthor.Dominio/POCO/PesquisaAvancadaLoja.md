# PesquisaAvancadaLoja
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaAvancadaLoja.cs

A classe `PesquisaAvancadaLoja` serve como um objeto de transporte de dados (DTO) utilizado para encapsular informações relacionadas à pesquisa avançada em lojas. Seu papel é facilitar a passagem de dados entre diferentes camadas da aplicação, especificamente no que tange às opções de pesquisa que o usuário pode definir.

## Propriedades
- `Nome`: Representa o nome da loja que será consultada.
- `Categoria`: Refere-se à categoria da loja, representada por um identificador numérico.
- `OpcaoVisualizacao`: Indica a opção de visualização selecionada para os resultados da pesquisa, provavelmente com implicações sobre a forma como os dados serão apresentados ao usuário.

Esta classe é anêmica e não contém lógica de negócio ou validações, funcionando unicamente como um armazenador de dados.
