# ConteudoFaixaHtml
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: ConteudoFaixaHtml.cs  

Esta classe é um DTO (Data Transfer Object) utilizado para o transporte de dados relacionados a conteúdos de faixas HTML em uma aplicação. Ela encapsula informações de estilo e estrutura de apresentação, sem lógica de negócios interna.

### Propriedades
- **CorFundo**: Representa a cor de fundo da faixa, que é utilizada para a estilização visual no frontend.
- **Colunas**: Uma lista de strings que contêm dados referentes às colunas que devem ser exibidas dentro da faixa HTML.

### Observações
Este DTO é utilizado para realizar a transferência de dados entre as camadas da aplicação, garantindo que as informações sobre a apresentação visual da faixa HTML cheguem corretamente à camada de visualização.
