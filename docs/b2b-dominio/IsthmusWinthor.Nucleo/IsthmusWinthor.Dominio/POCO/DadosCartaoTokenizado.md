# DadosCartaoTokenizado
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: DadosCartaoTokenizado.cs  

A classe `DadosCartaoTokenizado` serve para transportar dados relacionados ao cartão de crédito de um cliente, incluindo informações como o ID do cliente, o token do cartão, os últimos dígitos do cartão e a bandeira do cartão. 

---

### Observações

Esta classe é um DTO (Data Transfer Object) e não contém lógica de negócios ou validações, sendo responsável apenas por representar uma estrutura de dados.
