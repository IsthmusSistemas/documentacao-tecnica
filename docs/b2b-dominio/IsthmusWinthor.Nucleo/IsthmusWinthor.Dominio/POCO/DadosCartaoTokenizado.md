# DadosCartaoTokenizado
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: DadosCartaoTokenizado.cs  

A classe `DadosCartaoTokenizado` é um Data Transfer Object (DTO) que tem como finalidade transportar informações relacionadas ao cartão de crédito, incluindo o ID do cliente, o token gerado para transações, os últimos dígitos do cartão e a bandeira do cartão.
