# PesquisaAvancadaRemarketing
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaAvancadaRemarketing.cs  

A classe `PesquisaAvancadaRemarketing` serve como um objeto de transporte de dados que encapsula informações necessárias para realizar uma pesquisa avançada relacionada ao remarketing. Ela é projetada para transportar dados entre diferentes camadas do sistema sem incluir lógica de negócio.

### Visualização das Propriedades
- **DataInicial**: Representa a data inicial para a pesquisa.
- **DataFinal**: Representa a data final para a pesquisa.
- **SomenteCarrinhosAbandonadosAtualmente**: Indica se a pesquisa deve considerar apenas carrinhos que foram abandonados até o momento atual.

Essa classe é usada para facilitar a comunicação de parâmetros de pesquisa entre componentes da aplicação, assegurando que as informações necessárias para essa operação sejam facilmente acessíveis e transferíveis.
