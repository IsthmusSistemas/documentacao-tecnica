# PesquisaAvaliacaoProduto
**Namespace**: IsthmusWinthor.Dominio.POCO  
**Nome do Arquivo**: PesquisaAvaliacaoProduto.cs  

A classe `PesquisaAvaliacaoProduto` é um DTO (Data Transfer Object) que transporta dados relacionados à avaliação de produtos. Ela é utilizada para encapsular as informações que serão filtradas e analisadas com o objetivo de gerar relatórios e insights sobre a satisfação dos clientes em relação aos produtos.
