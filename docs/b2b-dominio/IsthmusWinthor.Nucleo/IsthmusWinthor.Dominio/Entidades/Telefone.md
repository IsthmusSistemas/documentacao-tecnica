# Telefone
**Namespace**: IsthmusWinthor.Dominio.Entidades  
**Nome do Arquivo**: Telefone.cs  

O `Telefone` é uma estrutura simples de dados utilizada para transportar informações relacionadas a números de telefone, especificamente o DDD (Discagem Direta à Distância) e o número em si.

--- 

### RESUMO
A classe `Telefone` atua como um Data Transfer Object (DTO), representando uma entidade básica que encapsula dados relacionados ao telefone de um usuário ou de um sistema, sem incluir lógica de negócios ou comportamentos complexos. É utilizada principalmente para transportar estas informações entre diferentes camadas da aplicação.

--- 

### Propriedades
- **Id**: Identificação única do telefone.
- **DDD**: Código de área do telefone.
- **Numero**: Número do telefone.

Trata-se de uma classe anêmica, não apresentando lógica de negócios, validações ou cálculos, concentrando-se apenas em estruturar dados em uma forma transportável.
