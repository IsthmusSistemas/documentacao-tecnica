# AlertaTemplate

- **Namespace**: IsthmusWinthor.Dominio.Entidades
- **Nome do Arquivo**: AlertaTemplate.cs

> AlertaTemplate é uma classe de transporte de dados para informações sobre alertas no sistema. Ela encapsula detalhes como distribuidora, tipo de canal, tipo de alerta, título e mensagem, facilitando a comunicação entre diferentes camadas da aplicação.
