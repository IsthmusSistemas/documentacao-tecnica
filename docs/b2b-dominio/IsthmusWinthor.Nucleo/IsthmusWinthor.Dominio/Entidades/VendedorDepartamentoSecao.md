# VendedorDepartamentoSecao
**Namespace**: IsthmusWinthor.Dominio.Entidades  
**Nome do Arquivo**: VendedorDepartamentoSecao.cs  

Esta classe serve como um Data Transfer Object (DTO) para transportar dados relacionados à associação entre um vendedor e suas atribuições dentro de um departamento e seção específicos no sistema. 

---

Nota: A classe não possui lógica de negócios ou regras definidas além de representar uma estrutura de dados.
