# ProdutoPrecoSankhyaResponse
- **Namespace**: IsthmusWinthor.Dominio.Sankhya.Model.Precos
- **Nome do Arquivo**: ProdutoPrecoSankhyaResponse.cs

Este é um DTO (Data Transfer Object) que serve para o transporte de dados relacionados ao preço de um produto no sistema Sankhya.

---

Como a classe é um DTO, não há lógica de negócio ou regras complexas a serem documentadas. Contudo, aqui está uma breve descrição de suas propriedades:

1. **Propriedades:**
   - `CodigoProduto`: Identificador do produto.
   - `CodigoLocalEstoque`: Identificador do local de estoque.
   - `Controle`: Informação adicional de controle.
   - `Unidade`: Unidade de medida do produto.
   - `Valor`: Valor do preço do produto.

---

Para referenciação futura, a classe servirá para estruturar dados com base em produtos e seus respectivos preços e pode ser utilizada em contextos onde a transferência de informações seja necessária entre diferentes partes do sistema.
