# PrecosSankhyaRequest
**Namespace**: IsthmusWinthor.Dominio.Sankhya.Model.Precos  
**Nome do Arquivo**: PrecosSankhyaRequest.cs  

A classe `PrecosSankhyaRequest` é um Data Transfer Object (DTO) que serve para transportar informações relacionadas a requisições de preços no contexto de vendas e negociações no sistema Sankhya. Ela encapsula os dados necessários para processar uma solicitação de preços, incluindo detalhes sobre o cliente, vendedor, e os produtos envolvidos na transação.
