# IPrecoSankhyaRepositorio
- **Namespace**: IsthmusWinthor.Infra.Sankhya
- **Nome do Arquivo**: IPrecoSankhyaRepositorio.cs

Este é um **DTO/ViewModel** que serve como contrato para a comunicação entre a aplicação e o repositório de preços.

---

## Notas
O código fornecido representa uma interface cujo propósito é buscar dados relacionados a preços de produtos para uma distribuidora específica. A interface contém um método que aceitaria um identificador de distribuidora e um objeto de requisição que contém informações sobre os preços.
