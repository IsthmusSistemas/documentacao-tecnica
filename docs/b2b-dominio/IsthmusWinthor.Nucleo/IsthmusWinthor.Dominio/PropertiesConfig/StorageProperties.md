# StorageProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: StorageProperties.cs  

A classe `StorageProperties` é um DTO (Data Transfer Object) utilizada para transportar dados relacionados à configuração de armazenamento em sistemas corporativos. Sua finalidade principal é encapsular informações como o nome do armazenamento, a string de conexão e a URL correspondente. Este tipo de classe não possui lógicas de negócios ou validações intrínsecas, servindo apenas para facilitar a transferência de dados entre diferentes partes do sistema.
