# SwaggerProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: SwaggerProperties.cs  

O `SwaggerProperties` é uma classe utilizada para transportar dados relativos à configuração do Swagger, que é uma ferramenta para documentação de APIs. Ela não contém lógica de negócio e seu propósito é fornecer propriedades que descrevem a API, sua aparência na interface do usuário e a versão da API web.

## Propriedades
- `Description`: Descrição da API que será exibida na documentação.
- `UIEndPoint`: URL do ponto de extremidade da interface do usuário do Swagger.
- `VersionWebApi`: Versão da Web API, normalmente no formato semântico (ex: "1.0.0").  

Esta classe simplesmente fornece um modelo estrutural para armazenar essas informações, permitindo que outros componentes do sistema acessem facilmente os dados de configuração do Swagger.
