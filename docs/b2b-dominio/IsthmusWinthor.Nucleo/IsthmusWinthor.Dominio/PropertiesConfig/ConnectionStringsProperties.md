# ConnectionStringsProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: ConnectionStringsProperties.cs  

A classe `ConnectionStringsProperties` é um DTO utilizado para transporte de dados relacionados a strings de conexão com o banco de dados. Sua finalidade é encapsular as informações de configuração de conexão, permitindo que outras partes do sistema acessem essas informações de forma organizada e isolada.
