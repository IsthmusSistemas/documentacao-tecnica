# OpenTelemetryProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: OpenTelemetryProperties.cs  

A classe `OpenTelemetryProperties` tem como finalidade atuar como um Contêiner de Propriedades para informações de configuração necessárias para a integração com o OpenTelemetry, uma ferramenta de observabilidade que coleta e monitora dados de desempenho de aplicações.

```markdown
> Nota: Esta classe é um DTO/ViewModel, utilizada exclusivamente para transporte de dados de configuração e não contém lógica de negócios.
```
