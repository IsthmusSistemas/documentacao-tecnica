# JWTProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: JWTProperties.cs  

Esta classe tem a finalidade de servir como um Data Transfer Object (DTO) para transportar propriedades associadas à configuração do JWT (JSON Web Token). Ela encapsula informações de login, senha e a chave utilizada para a geração do token, porém não contém lógica de negócio ou validações.
