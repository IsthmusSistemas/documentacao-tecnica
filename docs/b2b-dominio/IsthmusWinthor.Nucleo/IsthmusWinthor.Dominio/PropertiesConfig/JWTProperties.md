# JWTProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: JWTProperties.cs  

Esta classe é um Data Transfer Object (DTO) que serve como um modelo de transporte para as propriedades relacionadas à configuração de JSON Web Tokens (JWT). Ela contém as credenciais necessárias para autenticação e geração de tokens.
