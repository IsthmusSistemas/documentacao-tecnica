# GeneralProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: GeneralProperties.cs  

A classe `GeneralProperties` atua como um Data Transfer Object (DTO), responsável por armazenar e transportar as configurações gerais de um sistema, contendo URLs e parâmetros de acesso para diferentes serviços e ambientes. Ela serve principalmente como um meio para agrupar e estruturar esses dados, facilitando sua manipulação ao longo da aplicação.
