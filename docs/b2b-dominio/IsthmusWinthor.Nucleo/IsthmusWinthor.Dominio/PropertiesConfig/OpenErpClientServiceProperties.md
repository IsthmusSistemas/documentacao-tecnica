# OpenErpClientServiceProperties
**Namespace**: IsthmusWinthor.Dominio.PropertiesConfig  
**Nome do Arquivo**: OpenErpClientServiceProperties.cs  

A classe `OpenErpClientServiceProperties` é um Data Transfer Object (DTO) que serve para transportar informações de autenticação necessárias para que um serviço se identifique corretamente ao consumir outros serviços. Ela contém propriedades que são utilizadas para armazenar a URL do serviço, o ID do cliente e o segredo do cliente, essenciais para a autenticação.
