# TokenCartaoCielo
- **Namespace**: IsthmusWinthor.Dominio.EntidadeCartao.Cielo
- **Nome do Arquivo**: TokenCartaoCielo.cs

O `TokenCartaoCielo` é um Data Transfer Object (DTO) utilizado para o transporte de dados relacionados a informações de cartão de crédito junto à integração com o sistema da Cielo. Ele encapsula os dados necessários para efetuar transações financeiras, mas não contém lógica de negócio ou validação própria.
