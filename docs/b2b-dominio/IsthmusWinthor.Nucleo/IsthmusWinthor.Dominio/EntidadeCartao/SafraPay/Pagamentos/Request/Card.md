# Card
**Namespace**: IsthmusWinthor.Dominio.EntidadeCartao.SafraPay.Pagamentos.Request  
**Nome do Arquivo**: Card.cs  

A classe `Card` é um Data Transfer Object (DTO) que serve para transportar informações necessárias para a realização de pagamentos e transações financeiras. 

É responsável por encapsular dados relacionados a um cartão de crédito, incluindo informações do titular, detalhes do cartão e endereço de cobrança. Essa estrutura é fundamental para assegurar que as informações sejam transferidas de forma organizada e compreensível entre as diferentes camadas de um sistema corporativo.
