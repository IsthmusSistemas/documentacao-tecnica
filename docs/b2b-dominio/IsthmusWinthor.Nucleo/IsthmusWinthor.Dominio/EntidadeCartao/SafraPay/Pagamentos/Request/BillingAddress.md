# BillingAddress
**Namespace**: IsthmusWinthor.Dominio.EntidadeCartao.SafraPay.Pagamentos.Request  
**Nome do Arquivo**: BillingAddress.cs  

A classe `BillingAddress` é um Data Transfer Object (DTO) utilizada para o transporte de dados relacionados ao endereço de cobrança em transações de pagamento. Esta classe estrutura as informações necessárias para que um endereço de cobrança possa ser facilmente transmitido entre diferentes camadas de um sistema ou entre sistemas distintos. 

Como um DTO, `BillingAddress` não contém lógica de negócio ou validações. Sua finalidade principal é garantir a integridade e a organização dos dados referentes ao endereço.
