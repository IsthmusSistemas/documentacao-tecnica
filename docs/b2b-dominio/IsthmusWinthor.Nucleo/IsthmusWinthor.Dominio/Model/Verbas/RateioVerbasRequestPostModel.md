# RateioVerbasRequestPostModel
**Namespace**: IsthmusWinthor.Dominio.Model.Verbas  
**Nome do Arquivo**: RateioVerbasRequestPostModel.cs  

A classe `RateioVerbasRequestPostModel` atua como um objeto de transporte de dados (DTO), responsabilizando-se por encapsular informações necessárias para a requisição de rateio de verbas. Sua finalidade é transportar dados entre diferentes camadas da aplicação, assegurando que as informações relevantes sejam passadas de maneira organizada.
