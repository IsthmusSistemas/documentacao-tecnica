# EnderecoTipoEnum
- **Namespace**: IsthmusWinthor.Dominio.Enumeradores
- **Nome do Arquivo**: EnderecoTipoEnum.cs

O `EnderecoTipoEnum` é um enumerador que define os diferentes tipos de endereços que podem ser utilizados no sistema, permitindo a categorização adequada de endereços para funcionalidades específicas, como entrega, comercial e cobrança.
